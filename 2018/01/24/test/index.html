###移动端tap事件发生两次（或多次）

======
-原因导入了多个有tap事件包，如：MUI.js,MUI.min.js,zepto.js等

解决方式：

-1.当使用的是zepto和MUI时，可以订制一份没有touch事件的zepto包
-2.当使用tap事件时，使用setTimeout，让一定事件内只发生一次tap事件，代码如下：

​ $(select).on(‘tap’, function () {

​ if (!flag) {

​ flag = true;

​ count++;

​ console.log(count);

​ mui(‘#pageSlider’).slider().gotoItem(count);

​ setTimeout(function () {

​ flag = false;

​ }, 300);

​ }

​ });
